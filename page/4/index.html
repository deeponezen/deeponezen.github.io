目标固然重要，实现目标的人才是关键

发表于 2022-05-29 更新于 2024-11-24 分类于 时光集

「目标」

《老板不知道的我》里讲到一段：我觉得管理者，

  1. 不能用自己的目标吃掉大家的目标。

  2. 不能用结果的目标吃掉过程的目标。

  3. 不能用经济的目标吃掉情绪的目标。

我们经常听到，结果导向或者职场上情绪不重要，是这样的吗？

上面三个目标管理的观点，是我比较认同的，也是最近几年觉得非常值得思考和转变的目标管理方向。

结果重要吗，很重要，而忽略过程的事实求是，现在在我看来就是一场耍流氓。

10年前互联网的那个时代，大家都很拼，也遗留下了互联网的这种996恶习。我们应该不断地去花时间学习，怎么做好时间管理、精力管理，而不应该不断地去卷怎么延长工作时长，甚至做一些无效的工作。

很多事情并不能在规定时间很好解决的，要使视人为人，要考虑人性，要事实求是，要考虑境地，要不断总结实践和迭代改进。

项目任务管理中，不管敏捷、Scrum还是什么方法，我觉得一直最重要的两个点：

1.「反馈透明」：所有协作者应该对任务进度反馈有知情权。

  1. 「能力预估」：所有协作者应该对自我能力和合作者能力有预判，越接近越好。

如果这两个没做好，只追求结果就是一场无尽的噩梦，无限的死循环。

这些年我带过团队，开过公司，也可能是老了，喜欢做的两件事，一是很喜欢跟年轻人聊天，二是喜欢带新人，也愿意尽可能的给他们提供好的工作成长环境。

为什么这样做？一是我相信公司始终不是某个人的公司，不是老板的公司，不是中高管的公司，公司的首要目标是要服务客户和用户需求。二是，我相信未来是属于年轻人的，而老板还是高管，顶多只是一个角色，一个引路者，开拓者。

套用一句话，社会是人民的。

新人在成长的过程中，和团队和公司的目标就很难保持一致，这也就是OKR为什么要分层，不同层级的O是不一样的，但只要顶层的O是一致就没问题。

最后，情绪重要吗？很多时候我们太容易忽略人的感受了，如果我的工作因为太多一些条件资源不足，环境不公平，或者一些乱七八糟的事儿，我工作不愉快，我的投入肯定大大折损的。要平抚这些情绪需要消耗巨大的精力，甚至可能长久了会是一颗隐形的炸弹。

我们总是在考虑成本，显性成本很重要，但隐性成本也不能忽视，很多时候人才是公司的未来，而人能在开心顺畅的环境下工作生活才能最高的发挥能动性，也是降低成本的一种方式。

公司是需要发展的，未来是属于年轻人的。

目标固然重要，实现目标的人才是关键，让合适的人最大极限的完成目标才是关键。

是什么曾经拯救过你，你最好就试着用它来拯救这个世界

发表于 2022-05-27 更新于 2024-11-24 分类于 时光集

「时代·延续」

最近不管读书还是看的视频，都有很强烈的一种感觉，我们这代人所面临的、在解决问题，先辈们早在几百几千年都遇到过，也曾孜孜不倦得探索解决。

《十三邀·锺叔河》中，锺先生有一个观点也提到，他说：

“百年前的作品，我们现在还是那代人进程的继续，那个过程并没有终止，也并没有圆满地到达终点，它也不会有终点。他们有时候的一些苦恼，他们的一些摸索，他们的一些徘徊，对于我们现代的人还有直接的意义。”

读书的一个好处，是可以穿越时空，与不同时空的人交流，这是一件很神奇的事。

几百几千年前的知识智慧能传承下来，这或许是人类文明的庆幸吧，因为有了文字语言。

有时候还是觉得挺可悲的，互联网在解决信息闭塞的问题同时，也产生了无数的信息噪音。

如何去筛选并高效的学习优秀的知识呢，如何让优秀的知识分享并受用呢？

有那么一瞬间很想去百家争鸣的时代、苏格拉底的时代、文艺复兴的时代、爱因斯坦的时代、清末民初的时代、红色革命的时代等等，去看看时代的伟人是如何思考，如何去决策的，感觉那个时代的人有无尽的探索精神，有担当、有责任、有信仰。

《直击本质》的作者艾菲在书的序言写过一句话：是什么曾经拯救过你，你最好就试着用它来拯救这个世界。

什么是生，什么是死，也许对我来说真正的理解了生命的意义，生的意义才可能意识到自己是活着的，才能感受到该有的蓬勃生命力吧。

这世界谈不上你来拯救，被拯救的好像永远只是渺小的人类自己，如果这个世界真的有一天病了，希望有人能站出来的时候，这个人恰好你可以。

夫知者不言，言者不知

发表于 2022-05-20 更新于 2024-11-24 分类于 时光集

「天道·知道」

看完《天道》后，竟然无意中看到一段解读《庄子·知北游》，寓言故事大致如下：

一个叫「知」的问「无为谓」，怎么才算「知道」、「安道」、「得道」，无为谓不知道怎么答，然后知又去问「狂屈」，狂屈说知道，可到嘴边又没说；然后知去问「黄帝」，黄帝于是告诉知：无为谓是真知道，狂屈是接近了，而我和你终不得道。

天道的那句，“你知道你，所以你不是你”，应该是源自这里面的一句话：“夫知者不言，言者不知。”同样丁说芮的境地比他高，跟这则寓言的故事一样。

有意思的是，我读《知北游》的时候，对「知道」这个概念愣了一下，查了下百科竟然有5种意思：

  1. 谓通晓天地知道，神明人世之理。

  2. 认识道路。

  3. 晓得，谓对事物有所了解、认识。

  4. 公文用语。犹知悉，多见于诏令。

  5. 犹厉害。

想到日常我们用「知道」，是真的知道了吗？还只是一句口头禅，还只是「收到」、「听到」、「看到」等，真正有没有「知」，有没有理解其中的「道」？

至于「安道」，还是「得道」，都不重要了，像我这种人开了这扇门，一旦开始思考了、追寻了这个「道」，用庄子说的，终不得吧。

若干年前，读禅宗山水的三种境地，想始终无法明白第三种境地，更无从进入第三境地，想来，最高的境界还是：看山是山、看水是水。

自然至极，毫无雕琢，不可得。

你不知道你，所以你是你

发表于 2022-05-18 更新于 2024-11-24 分类于 时光集

「天道·境界」

今天把《天道》看完了，再次看的时候最后竟然有两个场景几近哭泣：

一个是芮小丹打算不去读律师，准备写小说，问了丁元英三个问题，丁一一解答，芮说当她决定想把「文化属性」通过小说等艺术的形式表达出来时候，有无尽的动力。丁举杯敬芮的「悟」的时候，我完全触动到了。

另外一个是芮牺牲后，丁对着芮给她的照片说到：“当生则生，当死则死，来去自如，丫头，不简单啊。”这个「不简单啊」，直接破防到哽咽。

丁以前跟芮说，她的境界比他高，芮不理解，觉得丁好像是有意抬高她，可丁真的是认为芮的境界比她高，甚至是丁不可达的境地。丁说他是想活明白，活得扭捏，而芮是率性洒脱。

丁能懂芮，芮能懂丁。

丁给芮说过了两句很玄的话：“你不知道你，所以你是你；如果你知道了你，你就不是你了。”

这话芮一直没明白，而芮也无需明白，也正是因为这个不知，芮才是芮。

天道里面讲丁很神，是透过文化属性，看透人性的人，是悟了道的人。可有时候悲哀的是，有人修得一生，曲曲折折，磕磕碰碰好像是得了道，甚至不可得，可有人却是不用修不用悟就得了道的人，丁是前者，芮是后者。

有人绕了一圈看山是山，看水是水，有人第一眼看山就是山，看水就是水。

不可得。

对自己有深刻的认知是一件很重要的事儿，活出自己也很重要，但很多人都是在追寻活出自己，而有的人就是在活出自己。

活出自己吧。

无所用，无所不用

发表于 2022-05-15 更新于 2024-11-24 分类于 时光集

「天道」

周末把《天道》电视剧又下来开始看了，今天看到关于「天道」的一段话：

———

丁元英说：“透视社会依次有三个层面：技术、制度和文化。小到一个人，大到一个国家一个民族，任何一种命运归根到底都是那种文化属性的产物。强势文化造就强者，弱势文化造就弱者，这是规律，也可以理解为天道，不以人的意志为转移。”

“强势文化就是遵循事物规律的文化，弱势文化就是依赖强者的道德期望破格获取的文化，也是期望救主的文化。强势文化在武学上被称为“秘笈”，而弱势文化由于易学、易懂、易用，成了流行品种。”

芮小丹说：“从字面上能理解一点，但知道又如何，怎么用呢？”

丁元英说：“无所用，无所不用。”

———

这个「无所用，无所不用」恰好是我这几天思想瓶颈的一个非常好的答案吧。

先说「天道」，这个可能就是我一直以来想寻找的规律，不以意志为转移的东西，可以是定律、思维、或者其他种种。

长久以来的探索，我确定并相信了《形式逻辑》这种底层驱动，以及对《马哲》这种世界观的认同。

确定逻辑和马哲作为底层驱动力之后，我迷惑了好些天，也许是苦苦探究的终于有了一个答案，也或许是终于达到阶段性的彼岸。

可即使掌握了形式，具体的内容还是需要突破的，我一度在想，要不要继续去研究具体的内容，但天下种种，哪怕有十足的好奇心和精力，也有终将耗尽的时候，想到这一层，那么一瞬间，顿感无趣和无力了，甚至失去了目标，也可能跟上海长时间遥遥无期的封闭有影响。

我迷惑的是，因为这些有种形而上，看似解决不了衣食住行的问题，不禁会问逻辑和哲学能带给我们什么呢？

芮小丹也提了非常好的这么一个问题：“有什么用呢？”

无所用，这个看起来没什么用，而后面一句「无所不用」恰巧是我一直以来相信的，只是没坚定下来的。

无所不用的边界就在于，逻辑和马哲其实是可以应用在日常生活工作的方方面面。

我跟朋友说，我30多岁了好像现在终于悟出了点门道，是幸运吧。

好在，再回过来，天道给了我一个明确的答案吧，其实答案自己早已知道，只是需要一个去相信和坚持的理由吧。

有个很有意思的问题：人类做很多事都需要找到某种意义，但这种意义真的重要吗？

直到现在我也不知道，但很多事对我来说是需要搞清楚为什么的。

所谓「无所用，无所不用」，其实是大道至简，看似简单的东西要做到其实很难，而那些复杂的东西，其实是可以很简单，只因为未知或把它想复杂了。

老子说：“无为而无不为”。

顺应「天道」，实事求是，见路不走，即见因果。

一阴一阳之谓道

发表于 2022-05-11 更新于 2024-11-24 分类于 时光集

「阴·阳」

古语云：“一阴一阳之谓道”。

有时候追求极致、念念不忘的思考好处就是证明自我的一个存在吧。

使用Flomo构建自己知识体系的时候经历了几个阶段：

  1. 经典摘抄，少感悟，随机关键词标签无分类。

  2. 多感悟，少摘抄，有分类标签[P.A.R.A]，结构化表达。

  3. 思考事实·观点、理性·感性、客观·主观、显性·隐性、自我·他人、个人·社会、有序·无序、正·反等等。

  4. 思考形式·内容，思考逻辑、追踪哲学、思考概念。

  5. 思考此岸·彼岸，思考大道至简、追求探索山水境界。

  6. 思考抽象·演绎。

  7. …

直到现在思考也谈不上到了那个心中追求的那个道的顶点，抑或能触碰到那个门的边界，每每阅读看到先辈们早在几百上千年就总结的精辟透彻，喜悦之极甚至感动的哭泣，倍感幸运吧。

《铿锵行天下第二季》有一个假设： 假如地球上没有了沙漠会产生怎样的后果？

好像沙漠恢复成森林是一件不错的壮举，但沙漠的存在带来是全球气候的一个循坏，如果沙漠变成了森林，那可能对地球就是一个新的生态灾难。

正义和罪恶是绝对的吗，对和错真的重要吗，黑与白中间其实是有灰的，有理数竟然不是终结，无理数有尽头吗？能量为什么会守恒的，熵值为什么会不断的增大等等。

计算机的世界不过是0和1，现实可能不过是一阴一阳，相互轮转，相互更迭。

也许真实和虚拟不过一念之间，什么又是那个量到质的边界点呢？

很多时候我们说的概念只不过是口头禅而已

发表于 2022-05-08 更新于 2024-11-24 分类于 时光集

「实事求是」

如果一句话讲来讲去，讲到最后不知道它是什么意思，这个可能是口头禅。

高中的时候，我们搞过一次辩论会，在具体的论题辩证的时候，双方一直在谈论世界观，最后突然对方同学说到，我们总是在说「世界观」，那请我方讲一些什么是世界观，我方一下懵了，没回答上来。当时是在教室里，全班同学都在，班主任全程也在旁听没说话，唯独这个问题问出来的时候，直接鼓掌叫好。

那次辩论会对我印象都很深刻，其中一个就是具体的这个问题，什么是世界观；另外一个就是，如果我们很多认知概念固意为很懂得很了解，张口就来，其实很多时候不过一句口头禅而已，没有用心，没有做深入的思考。

什么是实事求是？

《汉书·河间献王刘德传》记:“修学好古，实事求是。”

毛泽东也对「实事求是」做了一次讲解：“实事”就是客观存在着的一切事物，“是”就是客观事物的内部联系，即规律性，“求”就是我们去研究。

很多时候，我们在看待问题解决问题都是从主观意识出发，只凭经验感觉，没有调查，甚至依照毫无根据的信息来源。

什么是事实，什么是观点，什么是客观，什么是主观，只有基于这些客观的事实，我们才能理性的分析问题，解决问题，所谓的思考能力，批判思维的基础就是要基于事实的，我们总是讲乌合之众，有时候也讲集体智慧，这个都不重要，重要的是背后的结论是基于事实的，还是基于观点给的。

你理解的世界是什么样的，你相信的世界应该是什么样的，试着慢慢清楚这些，很多问题就迎刃而解了。

念念不忘，必有回响

发表于 2022-05-07 更新于 2024-11-24 分类于 时光集

「重生」

哲学有一个悖论问题，忒修斯之船，如果忒修斯的船上的木头被逐渐替换，直到所有的木头都不是原来的木头，那这艘船还是原来的那艘船吗？

有的说是，有的说不是，如果每天我们的细胞系陈代谢，有一天体内的细胞都替换了，我还是我吗？那如果我们每天思考认识都在更新，我们还是我们自己吗？

问题还在那儿，是不是原来的船不重要，新的生活总要继续。

如果说物质决定意识，意识作用于物质来说，一个新的认知或许会带来我们的一次「重生」。

英文「重」的词根是「re」，比如reborn、rework、rebuild、review、restart、recall等等，都是表示再、重新、重复的意思。

互联网里面经常说的一个机遇是，当新的一次科技生产力的出现，任何行业都可以被重新做一遍。

互联网从web1.0到web2.0，还有最近一直在炒的web3.0，大到各个行业都值得「RE」一遍。

那对于个人呢，我们很多认知是后天被灌输的，是从小到大的环境影响的，但这一些都不一定是对的，人类的习惯是行为决定的，行为是意识决定的。

那我们是不是值得把我们过往的知识、概念、认知都需要重新审视一遍呢？

也许这是一个漫长的过程，但一些重要的观念我们是值得审视的，最近几年一直在探索到底哪儿出了问题，直到最近似乎从两本书找到了源头有了答案，而且有意思的是这两本竟然都是教材，一本《形式逻辑》，另外一本是《马克思主义哲学》，也或许正是因为大学对这两门课有过好奇和接触，留了念想，才有今天追根溯源的一个路径吧。

简单的讲是三观的一次重新审视：世界观、价值观、人生观。

马哲里面讲到一个道理，人与世界的关系主要包括两方面：一是认识世界，二是改造世界。人类所有的活动不过都是围绕这两件事在做的，这也就是为什么现在我反过来看「雕作自我、普惠他人」是一个什么样的底层和价值驱动。先是认识自我、认识世界，然后是改造世界。

每个人都会有新生的机会，而这机会，需要的是对自己、对这个世界的一次重新审视，对三观的一次洗礼。

在我的价值观里面，只有对自己和世界的深刻认知，才能换来一个安心的彼岸之路。

思维的过程是一个运用概念、判断、推理的过程

发表于 2022-04-30 更新于 2024-11-24 分类于 时光集

「逻辑·形式逻辑」

我们总是在说逻辑，但什么是逻辑呢？

“逻辑”一词导源于希腊文λσγοS（逻各斯），原意指思想、理性、规律、语词等，日常说的逻辑有3种不同的意思：

第一，表示客观事物相互联系和发展的规律性，以及反映这种规律性的人的思维发展的规律性。例如，“中国革命的逻辑”、“事物的逻辑”。这里的“逻辑”都是指客观事物的规律性说的。又如，“作出合乎逻辑的结论”、“文章的逻辑性很强”。这里的“逻辑”就是指思维的规律性而言的。

第二，表示某种特殊的理论、观点或看问题的方法。比如，侵略者常常把对别国人民的侵略说成是“友谊”，对此，我们斥之为：“这是地地道道的强盗逻辑。”这里所说的“逻辑”，指的就是侵略者的这种颠倒是非黑白的特殊理论、观点或看问题的方法。

第三，表示研究正确思维的形式和规律的科学，即形式逻辑和辩证逻辑，而通常则习惯于用来表示形式逻辑。比如，“学点文法和逻辑”、“中学生要学点逻辑”。这里所说的“逻辑”就是指形式逻辑这门科学。

那什么是「形式逻辑」？

形式逻辑是研究思维的形式及其规律的，首先是一门研究思维的科学，恩格斯说，逻辑学是关于思维过程本身的规律的学说。

那什么又是「思维」，人们的思维活动是如何进行的？

毛泽东在《实践论》里有这样一段论述：人们在实践过程中，开始只是看到过程中各个事物的现象方面，看到各个事物的片面，看到各个事物之间的外部联系……这属于人们认识的感性阶段。在这个阶段里“人们还不能造成深刻的概念，作出合乎论理（即合乎逻辑）的结论”。但是，随着人们“社会实践的继续，使人们在实践中引起感觉和印象的东西反复了多次，于是在人们的脑子里生起了一个认识过程中的突变（即飞跃），产生了概念”。

前阵子对「概念」的源头也是模糊的，这段话也能很好地解释「概念」的由来。

思维的过程乃是对客观世界的一种概括性的间接反映过程，它具有概括性和间接性的特点，并且这种反映是借助于语言来实现的。

人们思维的过程都是一个运用概念、判断、推理的过程，所以概念、判断、推理就成为人们思维过程中用来反映客观现实所必不可少的基本形式，即逻辑学所说的思维形式。

———

以上的大部分内容来自华东师范大学哲学系逻辑学教研室编的《形式逻辑》，这本书已经出了第五版，我看过第一版，初稿竟然还是1963年完成的，第五版其实比第一版优化了很多。

我为什么对这本书感兴趣，最主要的一个问题是，我一直在思考怎么才能正确的理解这个世界吧。

形式逻辑注重的是思维的形式规律，对内容本身不关注，这个其实就是我一直在思考的，客观世界到主观世界的反映是不是有一个固定的范式，而逻辑在我现在看来就是底层了，也无外乎三样，概念、判断、推理。

科学也许不一定对，但起码是我现在笃定的一种世界观。

有时候想，中国教育对逻辑的普及教育真的太薄弱了，当我笃定了这条道路之后，未来有一天没准成为一个逻辑的布道者也不一定。

某种程度上讲，正确的理解世界就是在理解，我和世界的辩证关系，一个此岸到彼岸的螺旋之路、往复之路，一个意识和物质的相互作用之路。

习惯是个行为，行为更容易改变

发表于 2022-04-29 更新于 2024-11-24 分类于 时光集

「标签·定性」

前两天抖音上看到《此念官方》的一个心理学方面的短视频《什么样的话最不应该对孩子说》，很不错。

嘉宾黄仕明这样回答的：在身份层面负面的语言（不会说）。

举个例子，如果我有一个个案跟我说，嘿，老师，我是一个酒鬼，他找我做心理咨询。

我不会认同的，我会说，嘿，你是说，你有喝酒的习惯吗？

习惯是个行为，行为更容易改变。

如果这个人说，我是一个抑郁的人。

我会这样说，你是说，你笑得比别人少一点吗？

如果我的孩子跟我说，爸爸，我肯定会失败的。身份认同，我不可能成功。

我会说，噢，你是说，这一次，你会失败吗？

让那个绷紧的地方（深吐气），打开一个放松的空间，在这个空间里才可能有新的东西进来。

当然，明天考试可能还是会失败的，但是我们之间有一个同盟，一起有一个意图，看到未来如何可以做得比现在好一点点，我成为了孩子眼里那个同盟，相信我的那个人。

———

很多时候，不管是给自己或者给他人很容易打上性质的标签，比如，我是一个loser，他是一个坏人等等。

可能因为这一个性质的标签，接下来若干年都在背负着这个无形的压力，甚至让人透不过气来，如果幸运，能走出来很好，但如果走不出来，可能就一直是个死循环。

人生是没有边界的，是没有那么多标签的，更不应该定性，这个世界也不是非黑即白，非好即坏的，很多时候时间是最好的陪伴，积极的心态会改变很多的。
