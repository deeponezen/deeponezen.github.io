我之存在，幸运至极

发表于 2022-07-26 更新于 2024-11-24 分类于 时光集

「幸运」

我最近发现，自从公司来了00后小伙伴以后，开始逐渐自嘲老了。

好在自己内心总是觉得还年轻，一方面喜欢探索世界的新事物，保留着那仅有的一点好奇心，另外一方面也愿意跟他们聊天，了解他们的世界。

很多年了，我愈发觉得「幸运」真的太宝贵了，宝贵到觉察有些事有些人错过了就真的错过了，而遇见了，才是真的无比幸运。

也许正是因为自己好像走过很多弯路，才倍感觉察那些交点是可遇不可求的。

前段时间确定了两个理解世界的范式：一个是逻辑，确定是这个世界可以理解的；而另外一个是概率，很多时候结果不过是一个概率。

我现在也总是跟小伙伴们讲，可靠远比准确重要，这个世界很多时候没有百分百，当你确定有百分比之六十以上，其实就可以朝前进了。

我以前很难相信「缘分」之说，而现在不断地探索哲学背后，自己竟然也宁可相信有造物主一样。可惜，就目前认知边界，地球是人类了解到唯一有生命的星球，而自己恰巧在这文明之中竟能留下渺小的一点印记，着实是幸运的，而在这之下，你能遇见的人或者事，更是幸运至极的。

幸运好像对我来说逐渐变成一件奢侈品，我开始愈发关注自己的专注力，仿佛害怕人生太短，浪费掉那些大把大把的时间到那些无关紧要的事上面，又仿佛奢侈地愿意花掉无穷至毕生的时间在那些疗愈过自己的人或者事上面。

很多年前，创业之际，几个小伙伴围绕一个小办公桌上工作之时，一位同事看着书忽然流泪不止，我们竟也无所适从，不免疑问种种。

直到若干年后，自己读书之时，竟然也会因为某一句话，某一个概念，无论何时何地何种境地，疗愈之极，感动到泪流不止，忘我之境。

《傅佩荣西方哲学课》中讲赫拉克利特时，其中一条思想，万物皆逻辑。

当时，读到这句话直接的感受，原来有些事真的是可以超越时空的，也终于明白乔布斯愿意用一生的财富换取苏格拉底共处一个下午。

因为，突然你发现，如此复杂的世界，几千年之前，有人与你理解世界的方式竟如出一辙，顿时感动到一塌糊涂。

也不知道，有时候到底，因为无尽地探索这个世界的真实，才变得幸运；还是因为幸运，才开始探索这无尽世界的真实。

也许，概率本就是这世界的真相，我之存在，幸运至极。

朝闻道，夕死可矣

发表于 2022-07-02 更新于 2024-11-24 分类于 时光集

「对话」

最近在读《傅佩荣的西方哲学课》，已经第一遍完成10%了，这速度感觉真的要半年才能学完这课，不过读得确实比较细，收获也蛮多的。

很早就听过，乔布斯愿意用他一生的科技成就与财富，换取同哲学家苏格拉底共处一个下午。

起初的理解，苏格拉底很厉害，乔布斯也很厉害，乔布斯能从苏格拉底能获取到他觉得有价值的内容。

自开始探索认知领域之后，很多断断续续的感悟沉淀，终于在这套体系里面得到安抚，不过自己遇到的种种问题，先哲两千年也遇到过。

大抵人类的本性都是在试图证明自己的想法是正确的，如果一旦发现有同道中人，不免千杯逢知己，甚少，唏嘘感叹。

比如，读到苏格拉底之死竟然发现是为了捍卫法律，依然赴死，这简直太震撼了，这个生死已经超然了。

这也就终于明白乔布斯愿意用一生的财富与他来对话。

比如，在读到赫拉克拉特的时候，发现他的其中一个观点，他发现这个世界万物背后皆是逻辑。

这个我刚好是最近悟出的一点门道，感动至极，竟然这些先辈早已指出了方向。

比如，这些先哲可能逃离不开研究神学，经常也听到哲学背后是神学。

其实，这个神学，不是我们日常理解的「鬼神」，而在我看来就是「神即道，道法自然」，是这个世界宇宙运行的自然规律。

比如，关于这个世界的理解，不过最终都回归到人的三观上，所谓认知，不过就是三观的认知。

这点也是最近终于发现的一个结论，再次感动。

比如，哲学追求“低调的人生态度”，平安、平静、平衡。

这个也是我一直以来追求的心境，内心的一个平衡状态，只是迟迟不可得，好在哲学这条路也算给了我一个慰藉吧。

比如，笛卡尔说，“每一个人一生之中至少要有一次，要去怀疑所有能够被怀疑的东西。”

这个也是我理解的一种「重生」态度，一种感性到理性，生而为人的一个蜕变的过程。

孔子说：“朝闻道，夕死可矣。”

抽象的三个层次

发表于 2022-06-30 更新于 2024-11-24 分类于 时光集

「抽象」

最近给新同事培训复盘笔记的时候发现了一个普遍现象：大部分人在记笔记的时候都很容易一堆一堆地输出来，恨不得把听到的内容或者做得事儿挨个字噼里啪啦的全记下来。

结果就是，很难从笔记中发现重点，更不容易清晰地复述自己的内容。

这里面主要有个方法可以练习：那就是对自己的输出进行不断的提炼抽象分类。

刚好这两天也看到《傅佩荣的西方哲学课》讲到了三层抽象：

物理抽象

数学抽象

形而上抽象

这里面主要有一个我觉得思维比较有意思的地方，其实还是现实到意识，客观到主观的一个反映。

物理抽象，就是把它的物理特性、个别的东西去掉。

譬如你现在说：“我看到一辆车”。你看到车，你称它为车，但这辆车跟别的车不一样，因为天下没有两辆一样的车。你说这是车，就代表什么？你把物理特性去掉了。换句话说，物理抽象是人的理性的本能的运作。

一个小孩子看图片，看到一只狮子的画，在书上看起来跟猫差不多大小，他到动物园一看就知道那是一只狮子。为什么？他有物理抽象能力，他不会说：“这跟我书上看的平面的那个狮子不一样。”你不用担心，人作为人，他的理性就有这样的能力，叫做物理抽象。

你如果没有物理抽象的能力，你永远只能看个别性、个体性、特殊性，而不能说这是车，因为天下没有所谓的车，每辆车都不一样。

有时候我们经常说透过现象看本质，就是一种透过个性看共性的一种整体思维、宏观思维、系统思维的能力。

当我们谈论人类的时候，我们不是在谈论某一个具体的人。

数学抽象，我觉得可以用另外一个概念来表示，数学建模，就是对现实世界的一个代数几何的抽象。比如计算机的0/1信息表示、山水画简约画等等。

形而上抽象，不问这是什么东西，也不问能不能计算，只问“你存在吗？” 只要是存在的都是一样的，譬如说有一张桌子存在，还有太阳，还有月亮，还有房屋，它们都是一样的。

亚里士多德谈到形而上学的时候，他用一句话“把存在之物当作存在之物来看”，而不要看它是不是苹果，是不是圆形的东西，不要看那些，只看它是不是存在。

每一层抽象都是一个升维的过程，维度越高信息的压缩比其实更大，简练但涵盖的信息却更大，这也就是我们在经常记笔记容易忽略的。

我们如果更容易提炼抽象，也就更能把握事物的特性要点，进而可以推断普遍性。

逻辑学册

发表于 2022-06-25 更新于 2024-11-24 分类于 时光集

「逻辑·先行篇」

我们总希望看透事物的本质，弄清楚背后的规律，因为只有弄清楚了规律，就有可能知道怎么去应对了。

那什么是事物的本质呢？其实关键的一点就是逻辑。

逻辑这个词源自古希腊，音译是逻各斯，原意指思想、理性、规律、词语等，是思维的规则，是研究思维的形式及其规律的。

这里面有2个前置条件：

什么是「思维」，我们先简单地理解是客观到人大脑主观的一些列反应。

语言在表达逻辑的时候是不是有丢失？答案是可能有的。

我们先假设语言可以尽可能地表达逻辑本身，也有观点说逻辑离开语言是不能存在的，这个我们暂时不探讨，有专门研究这个领域的，《逻辑·语法·修辞》。

2000多年前，古希腊有个哲学家赫拉克利特就表达过，逻格斯就是躲在对立统一背后的那个隐藏的法则。

所谓对立统一，简单的说就是对立的万物是相互需要的，是不可分离的，是相互转化而合成一体的。

他认为即使宇宙一切的变化都是有法则可以依循的，这个力量就是逻格斯，就是对立统一背后的法则。

可能从我们懂事起、学生时、工作后，经常也都提到，逻辑能力很重要，但我们很多时候都容易把经验当作逻辑，真正的逻辑是值得推敲的，是可以辩证的。

逻辑主要两种方法归纳和演绎，但这两种方法都有问题，需要配合着使用。

归纳举个黑天鹅例子：

17世纪之前欧洲的人认为天鹅都是白色的，但随着澳大利亚的黑天鹅出来，这个不可动摇的观念瞬间崩溃了。

因为他们一直看到的天鹅都是白色的，他们就认为这个世界天鹅就只有白色的。

演绎举个经典推论例子：

所有人都会死，苏格拉底是人，所以苏格拉底会死。

这个推论过程是没有问题的，但前提可能会有问题。如果有人不会死呢，所有人都会死只是目前我们的认知经验得出的，或者苏格拉底具备人的所有特性，但他可能还会有超越我们人的属性，不是我们认为的人呢，比如外星人？这个演绎推论即使过程再准确，但可能得出的结论已经没有意义了。

再想想，我们日常是不是经常犯这些错误：

A这样做成功了，B这样做也成功了，所以我们这样做也一定能成功，是吗？

还有专家说、老板说、大佬说等等，都对吗？

很多时候，我们能成功的除了奋斗和能力本身之外，还可能有运气成份，只是被忽略了，而某种成功的经验可能会成为下次失败的绝对因素。

这个世界有很多定律，至少在现在科学认知是正确可以参考的，依据这些，我们可以不断地推敲追问日常很多的问题，直到推倒我们不能再推的「第一性原理」。比如：墒增定律，能量守恒定律，宏观世界的牛顿定律等。

我们再回过来思考一下，什么是逻辑，逻辑是我们思维的反应规律。

当然，我们日常说的逻辑通常有几种含义：

第一，表示客观事物相互联系和发展的规律性，以及反映这种规律性的人的思维发展的规律性。比如，事物的逻辑。

第二，表示某种特殊的理论、观点或看问题的方法。比如，他为人处事的逻辑我们很难理解。

第三，表示研究正确思维的形式和规律的科学，即形式逻辑和辩证逻辑，而通常则习惯于用来表示形式逻辑。这个地方就是我们经常提到的「形式逻辑」。

那么，形式逻辑是研究什么的呢？它是一门什么性质的科学呢？我们为什么要学习它呢？……对于这样一些问题，只有对这门科学有了比较系统的了解之后，才能完全弄清楚。

你连世界都没有观过，哪来的世界观

发表于 2022-06-24 更新于 2024-11-24 分类于 时光集

「三观：世界观·人生观·价值观」

《后悔无期》中有一句台词：“你连世界都没有观过，哪来的世界观？”

哲学是有关世界源头的，是在反思世界真实性的，哲学的发展是人类从感性上升到理性思考的过程。

世界观简单的讲就是「你」对世界的看法，但这个「你」，在你我他当中就千差万别，每个人的理解都可能不一样，其次是深信的都不一样。

好比，有的人相信神明，有的人相信科学等，在无法用理性逻辑思考的事物出现之时，感性的认知和假设也会占据了主导。

世界观本就无所谓对错，可能有的不过是道不同。

因为对世界的认识和感知后，对人生以及价值就有了判断和追寻。

每个人都会想着过着属于自己，或者某种觉得有意义的生活，对世界的善恶、美丑、是非、对错也有了分辨。

很多时候，世界观才是你人生轨迹的源头，你相信的世界是什么样的，你追求假设的世界是什么样的，你的人生目标，以及价值输入输出就有了定性，路径其实也就在某个范畴边界了。

但好在世界观是动态的，你的认知是可以无限否定、可以无限升级的。

唯有不断的拓宽自己的认知边界，也许才可能触及到这个世界本来的真实性吧。

知人者智，自知者明

发表于 2022-06-19 更新于 2024-11-24 分类于 时光集

3000多年前，希腊德尔斐神庙阿波罗神殿门前有三句石刻：

认识你自己。「ΓΝΩΘΙ ΣΑΥΤΟΝ | Know thyself」

凡事勿过度。「ΜΗΔΕΝ ΑΓΑΝ | Nothing in excess」

妄立誓则祸近。「ΕΓΓΥΑ, ΠΑΡΑ ΔΑΤΗ | Surety brings ruin」

德尔斐神谕被称之为希腊人的精神支柱，有说，出自苏格拉底。

苏格拉底就认为自己是无知的，这种无知是对权威的结论有所警惕，保持一种质疑的心态，但「神明」说，没有人比他更聪明了。(此处神明说，是神殿祭司代传神明话语。)

他为此还特意求证了自己的无知，去拜访了三种人：

第一种是领导城邦的政治人物；

第二种是许多粉丝的作家；

第三种是拥有专业技术的工匠。

最后发现，这三种人可能在自己的领域都非常优秀，也因为这些，这些人很容易觉得自己非常聪明，什么都懂，并且在很多别的事情上发表许多见解，甚至带节奏。

苏格拉底最后的结论：神明说我明智，可能就是因为我知道自己是无知的；别人都跟我一样无知，但他们自以为有知，在自己本行以外的事物也以为自己胜过别人。

年少的时候，对世界是一种天然的无知，带来的是盲目的自信甚至自负，反而随着经历阅历种种，读的书慢慢多了，越发觉得自己是真正的无知，这种无知是一种对宇宙的敬畏，感知自己无比渺小的一种谦卑，沧海一粟而已。

人很难对自己有清晰的认知，首先可能很多人需要很长一段时间，才逐渐真正考虑认识自己，反观自己，思考自我；其次，即使开始自我认识，但不一定真正懂得了自我。这个过程在我看来，其实是伴随着人的一生。

古语有云，”人贵在有自知之明“，源自《道德经》，“知人者智，自知者明”。

又想到丁元英对小丹的那种有知的无知，无比地羡慕。

生活工作中，要相信专业，但也要敢于批判；读书也一样，尽信书不如无书。

终有一天，你也会和那些大人物一样，进入哲学的领地

发表于 2022-06-12 更新于 2024-11-24 分类于 时光集

「哲学·开篇」

我原以为还需要一段时间，才会系统的开启一段哲学之旅，要等到对这个世界的理解有了一定的了解和积累，现在提前了，总要开始的。

当时决定写小报童，也只是想把日常的一些思考沉淀下来，并且把一些觉得不错的Flomo沉淀，再以某种仪式感，刚好通过小报童输出出来。坚持了三个月下来，效果真的挺不错，很多沉淀思考脉络也越来越清晰。

这一年终于悟出了点门道，今天在精读《傅佩荣的西方哲学课》里面有这么一句话：

“不管你有没有摆脱柴米油盐，只要你有高级的灵魂，终有一天，你也会和那些大人物一样，进入哲学的领地”。

我庆幸是幸运的。

我并不觉得自己有高级的灵魂，但终于有一天自觉到，原来哲学是可以，给我解决我的人生问题指了条道，这就是万幸的，余生还有大半生，感动至极，幸运至极。

花了大半年的时间，研究逻辑学，总是逃不开哲学，因为逻辑学是哲学的基础，哲学是逻辑学的上层。

现在发现，活到现在，很多问题不过是一个世界观的问题：

这世界，你到底是如何理解的？

有了世界观，自然就有了人生观和价值观，也逐渐发现，太多太多问题的发源动机，无外乎三观逻辑，理解了这些，自上而下很多问题就很好理解并游刃有余了。

但人类记载文明2000多年一来，人类进步的可能只有科技，但本源问题，思维意识等直到现在也没有多大突破，现在很多研究也不过是先哲思考探索的延续。

生而为人，视人为人

发表于 2022-06-12 更新于 2024-11-24 分类于 时光集

「视人为人」

我们都是稀里糊涂的被动的就来到了这个世界，自记事起，又被传授了无数不知道对与错的经验和知识，最终也会被打上一个标签，人。

人类有文字记载的文明不过几千年，这几千年都不断地在研究一个问题，人到底是什么？中国儒释道文化最终不过也是讲了这些。

这个世界有善也有恶，有好人也有坏人；性本善，还是性本恶呢？人类的七宗罪是不是就是人类文明超脱的一个桎梏呢？

这是一种世界观、人生观、价值观的表现，我们大多数人不过是想着人性都应该善，人人生而自由平等，世界永远和平，这是一种向往和为之奋斗的目标，但有一部分人可能不是这样想的。

生而为人，怎么才算人呢？ 人生不过两件事：做事，为人。

事情有时候已经很难做了，为人可能更难了，古往今来又有多少人活得明白和通透呢？

人的属性其实有太多，这些属性一部分是人性的自然属性，生理结构所反应的，区别与其他物种。

另外一部分是社会属性，是文明发展的一种共同行为约定准则，是道德、是法律。

人的自然属性和社会属性，每一个人都会选择性遵守和相信的，就直接影响到他的三观准则。

《一代宗师》里面讲到三个境界：见自己，见他人，见众生。

你觉得自己是什么样的人呢，你觉得他人是什么样的呢，这个世界又是什么样的呢，他们可能是什么样的，他们应该又是什么样的呢？

物种的多样性一定要够，不能单极化

发表于 2022-06-05 更新于 2024-11-24 分类于 时光集

「多样性」

今天把《圆桌派·尹烨》又看了一遍，信息量很大，择其中一个视角分享一下：

“物种的多样性一定要够，不能单极化。比如番茄，如果全世界只栽培一种番茄，只有栽培种，没有野生种，那么只要这个病毒感染一次，这个就没了。”

《星际穿越》里面有一个场景，偌大的玉米地也是因为一次病毒就全没了。

人类基因已经测序完毕，总共30亿个碱基对，但现在具体哪个碱基对控制什么还不完全清楚。

生物的多样性能很好的保证物种的延续性，基因的自私性也可能意味着抵抗力的脆弱性，这也能很好的解释为什么「共生」才应该是生态的最终选择。

比如，很多对抗很多疾病病毒，我们都是从一些植物、动物种提炼出来然后研制成药或者疫苗才得以人类对抗，这是幸运的。

如果有一天只有人类了，动植物的生态多样性单一化，也就意味着人类的灭亡。

其实对于人类文明及个体也是一样，如果一味地只是个体文明的传达，没有开放的包容和学习态度，自然会被时代淘汰掉。
